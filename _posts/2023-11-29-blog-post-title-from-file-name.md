## How to change font size in VSCode - short guide
- Use the shortcut CTRL + SHIFT + P and start typing 'User settings'
<a data-flickr-embed="true" href="https://www.flickr.com/photos/197764307@N08/53363234631/in/dateposted-public/" title="vscode-ctrl-shift-p-user-settings"><img src="https://live.staticflickr.com/65535/53363234631_855d4077c1_c.jpg"/></a>
- Hit ENTER
- You should see an option called font size under the 'Commonly Used' section:
<a data-flickr-embed="true" href="https://www.flickr.com/photos/197764307@N08/53363553604/in/dateposted-public/" title="vscode-settings-fontsize"><img src="https://live.staticflickr.com/65535/53363553604_32893a306f_k.jpg"/></a>
- By default, the font size is 14. I changed it to 18 in this example.
- You can click on the font size field and type which value you want.
- You don't need to save any changes, they're applied as soon as you change the value.
- In this same interface, you can change the font family, the tab size (how many spaces equal a TAB), the cursor design.
- Finally, close the settings tab by left-clicking the X button.
